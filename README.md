# REITs Announcement Monitor

Monitor SSE ( Shanghai Stock Exchange ) and SZSE ( Shenzhen Stock Exchange ) REITs announcements.

## Features
- 6136+ announcements from 2021-2026
- Filter by exchange, category, date range, keyword
- Pagination with 20 items per page
- New announcement notifications
- Server-side rendered, zero JavaScript
- Auto-refresh every 3 minutes

## Deploy to Render

1. Fork/upload this repo to GitHub
2. Go to [Render.com](https://render.com)
3. Create new Web Service → Connect your repo
4. Build Command: 
pm install
5. Start Command: 
pm start
6. Deploy!

## Environment Variables
- PORT - Server port (Render sets this automatically)
